Noções Comuns

O que Euclides chama de "noção comum" são equivalentes aos axiomas no meio acadêmico atual. Elas tratam de afirmações consideradas verdadeiras e inquestionáveis que são usadas como base para provas futuras, sendo que elas mesmas naão precisam ser provadas.
  1. Coisas iguais a uma outra coisa são iguais entre si.
  2. Se iguais são adicionados a iguais, os resultados são iguais.
  3. Se iguais são retirados de iguais, os restos são iguais.
  4. Coisas que coincidem com uma outra são iguais entre si.
  5. O todo é maior que as partes.

Uso das noções comuns

Como podemos notar, todos os axiomas utlizados por Euclides tratam de magnitudes, definindo igualdade e operações dentro do livro. Um ponto importante a se notar é que primeiro é estabelecida a igualdade para então se definir a soma e a sibtração dentro destas igualdades, isso se deve ao fato de não fazer sentido operações com coisas deiferentes como uma linha e um retângulo, embora exista a possibilidade de comparações entre polígonos por exemplo.

Além disso, existem outras propriedades não listadas nas noções comuns que são usadas durante o livro, embora algumas delas possam ser tidas como consequência lógica delas.

Os Elementos, embora desenvolvido com base na lógica, algumas vezes não segue normas da matemática moderna. Na maioria das vezes destacaremos onde isso acontece e qual seria a forma moderna.
